## Deutsch

### Stabilitätsupdate für große Einsortierungen

- Thor ROM Butler protokolliert laufende Dateioperationen jetzt in einem
  kleinen Transaktionsjournal. Wird die App während einer großen Extraktion
  unerwartet beendet, erkennt und entfernt der nächste Versuch die
  unvollständigen Archivdaten sofort.
- Lose ROMs bleiben ebenfalls geschützt: Wurde eine Quelldatei im kritischen
  Moment bereits temporär verschoben, stellt die App sie anhand des Journals an
  ihrem ursprünglichen Ort wieder her.
- Der Speicherplatzschutz wurde mit absichtlich zu wenig freiem Speicher
  geprüft. Der Start bleibt gesperrt, bevor eine große Datei das Laufwerk
  füllen könnte.
- Zusätzlich getestet: 4.641 Funde in einem Ordnerlauf, beschädigte ZIPs,
  nicht unterstütztes RAR5, schnelle Wiederholung, App-Wechsel, Drehung,
  Display aus sowie eine vollständig gestreamte 3-GB-ZIP-Extraktion.

### Installation

`ThorROMButler-v1.9.1.apk` über die bestehende Installation installieren.
Einstellungen, eigene Systeme, ROMs, Sicherungen, Lernstand und Log bleiben
erhalten.

## English

### Stability update for large sorting runs

- Thor ROM Butler now records active file operations in a small transaction
  journal. If the app is unexpectedly interrupted during a large extraction,
  the next attempt detects and removes incomplete archive data immediately.
- Loose ROMs remain protected as well: if a source file had already been
  staged during the critical interruption window, the app restores it to its
  original location using the journal.
- The storage guard was verified with deliberately insufficient free space.
  Starting remains blocked before a large file could fill the volume.
- Also tested: 4,641 findings in one folder scan, damaged ZIPs, unsupported
  RAR5, rapid rescans, app switching, rotation, screen off, and a fully
  streamed 3 GB ZIP extraction.

### Installation

Install `ThorROMButler-v1.9.1.apk` over the existing app. Settings, custom
systems, ROMs, backups, local learning data, and the action log are preserved.
