## 🇩🇪 Deutsch

### Neu
- **⚡ Schnelleinstellungs-Kachel für den LAN-Empfang**: Statusleiste ausklappen → Stift-Symbol → „LAN-Empfang" hinzufügen. Ein Tipp startet/stoppt den Empfang, ohne die App zu öffnen — bei aktivem Empfang zeigt die Kachel direkt die Upload-Adresse für den PC-Browser.

### Sonstiges
- Abhängigkeiten aktualisiert (XZ 1.12), schnellere Builds durch Gradle-Configuration-Cache.
- Neues Social-Preview-Bild fürs Repo.

### Installation
`ThorROMButler-v1.1.0.apk` über die bestehende Installation installieren — Einstellungen bleiben erhalten.

## 🇬🇧 English

### New
- **⚡ Quick Settings tile for LAN receive**: expand the notification shade → pencil icon → add "LAN receive". One tap starts/stops receiving without opening the app — while active, the tile shows the upload URL for your PC browser.

### Other
- Dependency maintenance (XZ 1.12), faster builds via Gradle configuration cache.
- New social preview image for the repo.

### Install
Install `ThorROMButler-v1.1.0.apk` over the existing app — settings are kept.
