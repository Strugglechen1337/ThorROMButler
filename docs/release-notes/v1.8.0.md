## Deutsch

### Der Bibliotheks-Doktor

- Ein neuer eigener Bildschirm bündelt Sammlung, Fehlzuordnungen, Varianten,
  BIOS-/DAT-Bereitschaft und den Zustand der ROM-Sicherung.
- M3U-Playlists und CUE-Dateien werden auf fehlende, leere oder unlesbare
  Verweise geprüft. Gepackte Arcade-/Neo-Geo-ZIPs werden ohne Entpacken über
  ihr Dateiverzeichnis kontrolliert.
- Befunde haben eine Detailansicht und können für den aktuellen Lauf
  ausgeblendet werden. Eindeutig falsch einsortierte ROMs öffnen die bekannte
  Review-Reparatur; eine fällige konfigurierte Sicherung startet direkt.
- Die optionale SHA-256-Tiefenprüfung und ein neuer zweisprachiger
  Markdown-Gesundheitsbericht runden den Check ab.
- Arcade- und Neo-Geo-ZIPs werden jetzt korrekt in Statistik und
  Duplikatprüfung gezählt. Automatisch gelöscht wird weiterhin nichts.

### Installation

`ThorROMButler-v1.8.0.apk` über die bestehende Installation installieren.
Einstellungen, eigene Systeme, Sicherungen und Log bleiben erhalten.

## English

### The Library Doctor

- A new dedicated screen combines the collection, misplacements, variants,
  BIOS/DAT readiness, and ROM backup status.
- M3U playlists and CUE sheets are checked for missing, empty, or unreadable
  references. Packed Arcade/Neo Geo ZIPs are inspected through their file
  directory without extraction.
- Findings have a detail view and can be hidden for the current run. Clearly
  misplaced ROMs open the familiar review repair; a due configured backup can
  be started directly.
- The optional SHA-256 deep scan and a new bilingual Markdown health report
  complete the check.
- Arcade and Neo Geo ZIPs now count correctly in statistics and duplicate
  scans. Nothing is ever deleted automatically.

### Installation

Install `ThorROMButler-v1.8.0.apk` over the existing app. Settings, custom
systems, backups, and the action log are preserved.
