## Deutsch

### CISO-Erkennung für GameCube und Wii

- CISO-Dateien werden jetzt automatisch als GameCube- oder Wii-Spiel erkannt.
- Das funktioniert auch, wenn die CISO-Datei in ZIP, 7z oder RAR4 verpackt ist.
- Die App prüft sowohl den CISO-Container als auch die Disc-Kennung und lädt
  dafür nicht das vollständige Spiel in den Arbeitsspeicher.
- Ist die Datei ungültig oder nicht eindeutig, bleibt die manuelle Zuordnung
  erhalten. Es wird weiterhin nichts ohne Bestätigung einsortiert.

### Installation

`ThorROMButler-v1.4.2.apk` über die bestehende Installation installieren.
Einstellungen, eigene Systeme und Log bleiben erhalten.

## English

### CISO detection for GameCube and Wii

- CISO files are now detected automatically as GameCube or Wii games.
- This also works when the CISO file is packed inside ZIP, 7z, or RAR4.
- The app validates both the CISO container and embedded disc marker without
  loading the complete game into memory.
- Invalid or ambiguous files still use manual assignment. Nothing is sorted
  without confirmation.

### Installation

Install `ThorROMButler-v1.4.2.apk` over the existing installation. Settings,
custom systems, and the action log are preserved.
