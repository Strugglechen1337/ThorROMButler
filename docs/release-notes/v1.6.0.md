## 🇩🇪 Deutsch

### Der Butler im neuen Anzug ⚡

- **Neue Display-Schrift**: Alle Überschriften tragen jetzt Chakra Petch (SIL OFL) — ein leicht technischer, eckiger Schnitt, der zur Blitz-Identität passt. Fließtexte bleiben in der bestens lesbaren Systemschrift.
- **Aufgeräumte Einstellungen**: Lange Erklärtexte verschwinden hinter dezenten ⓘ-Icons und klappen bei Bedarf sanft auf. Jede Karte ist auf einen Blick erfassbar — nichts geht verloren.
- **Federnde Animationen**: Screens gleiten mit Spring-Physik herein, der Gamepad-Fokus poppt sanft statt hart zu rahmen, und nach einem fehlerfreien Einsortieren feiert ein goldener Blitz: „N ROMs einsortiert!"
- **Neue Scan-Momente**: Ein pulsierender Blitz begleitet den Scan, und der „Alles aufgeräumt"-Zustand zeigt ein sanft schwebendes Blitz-Badge im Glow-Kreis.
- Modernisierte Formen-Skala (größere, bewusst variierende Eckenradien) in allen drei Farbwelten — Thor, Odin und CRT erben alles automatisch.

Keine funktionalen Änderungen — Touch und Gamepad verhalten sich exakt wie vorher.

### Installation

`ThorROMButler-v1.6.0.apk` über die bestehende Installation installieren — Einstellungen bleiben erhalten.

## 🇬🇧 English

### The butler's new suit ⚡

- **New display font**: all headlines now wear Chakra Petch (SIL OFL) — a slightly technical, squared cut that matches the lightning identity. Body text stays on the highly readable system font.
- **Decluttered settings**: long explanations tuck behind subtle ⓘ icons and unfold smoothly on demand. Every card is scannable at a glance — nothing is lost.
- **Springy animations**: screens glide in with spring physics, the gamepad focus pops gently instead of hard-framing, and a golden bolt celebrates a clean sorting run: "N ROMs sorted in!"
- **New scan moments**: a pulsing bolt accompanies the scan, and the all-tidied-up state shows a softly floating bolt badge in a glow circle.
- Modernized shape scale (larger, deliberately varied corner radii) across all three color themes — Thor, Odin, and CRT inherit everything automatically.

No functional changes — touch and gamepad behave exactly as before.

### Install

Install `ThorROMButler-v1.6.0.apk` over the existing app — settings are kept.
