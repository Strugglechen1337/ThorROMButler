## 🇩🇪 Deutsch

### Neu

- **🧰 BIOS-Einsortierung**: Der Butler hat BIOS-Dateien bisher nur höflich aus der ROM-Liste herausgehalten — jetzt räumt er sie auch weg. Erkannte BIOS-/Firmware-Dateien (PS1 `scph*`, `gba_bios.bin`, NDS, Dreamcast, Amiga Kickstart, PC-Engine syscard, …) bekommen im Scan eine eigene Karte und wandern mit einem Tipp in deinen BIOS-Ordner.
- **Frei wählbares Ziel**: Der neue optionale BIOS-Ordner in den Einstellungen (neben dem DAT-Ordner) zeigt auf den System-Ordner deines Emulators oder wohin auch immer deine BIOS-Sammlung gehört.
- **Sicher wie immer**: Nichts wird überschrieben (Kollisionen bekommen ein `(1)`-Suffix), der ganze Lauf steht als eine Aktion im Log und lässt sich mit einem Tipp rückgängig machen.

### Installation

`ThorROMButler-v1.7.0.apk` über die bestehende Installation installieren — Einstellungen bleiben erhalten.

## 🇬🇧 English

### New

- **🧰 BIOS sorting**: so far the butler only politely kept BIOS files out of the ROM list — now he shelves them too. Detected BIOS/firmware files (PS1 `scph*`, `gba_bios.bin`, NDS, Dreamcast, Amiga Kickstart, PC Engine syscard, …) get their own card in the scan and move to your BIOS folder with one tap.
- **Freely chosen target**: the new optional BIOS folder in Settings (next to the DAT folder) points at your emulator's system folder or wherever your BIOS collection lives.
- **Safe as always**: nothing is overwritten (collisions get a `(1)` suffix), the whole run is logged as one action and can be undone with one tap.

### Install

Install `ThorROMButler-v1.7.0.apk` over the existing app — settings are kept.
