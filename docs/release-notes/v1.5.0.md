## 🇩🇪 Deutsch

### Neu

- **💾 ROM-Sicherung mit Wiederherstellung**: Einstellungen → „ROM-Sicherung" — Zielordner wählen (SD-Karte, USB, beliebiger Ordner) und „Jetzt sichern". Der Butler spiegelt das **komplette ROM-Verzeichnis mit allen Systemen** als ganz normale 1:1-Ordnerkopie, die jedes Frontend direkt lesen könnte.
- **Inkrementell**: Folgeläufe vergleichen den Bestand und kopieren nur, was neu ist — die zweite Sicherung einer 100-GB-Sammlung dauert Sekunden statt Stunden.
- **Wiederherstellen ohne Risiko**: Fehlende Dateien kommen aus der Sicherung zurück; was in der Bibliothek existiert, wird **grundsätzlich nie überschrieben**.
- Läuft als Vordergrund-Dienst mit Fortschritts-Benachrichtigung und Abbrechen (bereits Kopiertes bleibt erhalten, der nächste Lauf setzt fort). Freier Speicherplatz wird vorab geprüft, und die Einstellungen zeigen Datum, Umfang und Größe der letzten Sicherung.

### Installation

`ThorROMButler-v1.5.0.apk` über die bestehende Installation installieren — Einstellungen bleiben erhalten.

## 🇬🇧 English

### New

- **💾 ROM library backup & restore**: Settings → "ROM library backup" — choose a target (SD card, USB, any folder) and tap "Back up now". The butler mirrors the **complete ROM folder with all systems** as a plain 1:1 folder copy that any frontend could read directly.
- **Incremental**: follow-up runs diff the contents and only copy what is new — the second backup of a 100 GB collection takes seconds instead of hours.
- **Risk-free restore**: missing files come back from the backup; anything that exists in your library is **never overwritten, period**.
- Runs as a foreground service with progress notification and cancel (completed files are kept, the next run continues). Free space is checked upfront, and Settings show date, file count, and size of the last backup.

### Install

Install `ThorROMButler-v1.5.0.apk` over the existing app — settings are kept.
