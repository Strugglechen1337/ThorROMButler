## Deutsch

### Der Sammlungs-Assistent

- Thor ROM Butler lernt jetzt lokal aus manuellen Systemzuordnungen, deren
  Einsortierung erfolgreich abgeschlossen wurde. Fehlgeschlagene oder
  abgebrochene Vorgänge zählen nicht.
- Ähnliche unbekannte Dateien erhalten später einen klar gekennzeichneten
  „Gelernten Vorschlag". Er bleibt gelb, wird nie automatisch ausgewählt und
  braucht weiterhin deine Bestätigung. Gleich starke Widersprüche erzeugen
  keinen Vorschlag.
- Das Lernen kann in den Einstellungen pausiert und der lokale Lernstand
  vollständig gelöscht werden. Der Einstellungs-Export sichert ihn mit.
- Der Bibliotheks-Doktor empfiehlt bei mehreren Regionen oder Revisionen
  unverbindlich eine Variante anhand von Gerätesprache, Region, Dump-Qualität
  und Revision. Gründe stehen in den Details und im Markdown-Bericht.
- Es wird weiterhin nichts automatisch verschoben oder gelöscht.

### Installation

`ThorROMButler-v1.9.0.apk` über die bestehende Installation installieren.
Einstellungen, eigene Systeme, ROMs, Sicherungen und Log bleiben erhalten.

## English

### The Collection Assistant

- Thor ROM Butler now learns locally from manual system assignments whose
  sorting operation completed successfully. Failed or cancelled operations do
  not count.
- Similar unknown files may later receive a clearly labelled "Learned
  suggestion". It stays gold, is never selected automatically, and still
  requires confirmation. Equally strong conflicts produce no suggestion.
- Learning can be paused in Settings and all local learning can be cleared.
  Settings export includes it.
- For multiple regions or revisions, Library Doctor can recommend a variant
  based on device language, region, dump quality, and revision. Reasons appear
  in details and the Markdown report.
- Nothing is ever moved or deleted automatically.

### Installation

Install `ThorROMButler-v1.9.0.apk` over the existing app. Settings, custom
systems, ROMs, backups, and the action log are preserved.
