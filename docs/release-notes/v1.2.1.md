## Deutsch

### LAN-Empfang unter Android 17 repariert

- Android 17 blockiert lokale Netzwerkverbindungen bei Apps mit aktuellem
  Ziel-SDK standardmäßig. Thor ROM Butler fragt die dafür erforderliche
  Freigabe jetzt genau dann an, wenn der LAN-Empfang gestartet wird.
- Der Start funktioniert sowohl aus den Einstellungen als auch über die
  Schnelleinstellungs-Kachel.
- Wird die Freigabe nicht erteilt, bleibt der LAN-Server aus. Update-Check,
  Scan und Einsortieren benötigen diese Berechtigung nicht.
- Die zufällige Sitzungsadresse und das automatische Ende nach 30 Minuten
  bleiben unverändert. Ein neuer automatischer Upload-Test sichert den
  vollständigen Übertragungsweg ab.

### Installation

`ThorROMButler-v1.2.1.apk` über die bestehende Installation installieren.
Einstellungen, eigene Systeme und Log bleiben erhalten. Beim ersten Start des
LAN-Empfangs unter Android 17 die Abfrage für das lokale Netzwerk erlauben.

## English

### Fixed LAN receive on Android 17

- Android 17 blocks local network connections by default for apps targeting
  the current SDK. Thor ROM Butler now requests the required permission exactly
  when LAN receive is started.
- Starting receive mode works from both Settings and the Quick Settings tile.
- If access is denied, the LAN server remains off. Update checks, scanning, and
  sorting do not need this permission.
- The random session address and automatic 30-minute timeout remain unchanged.
  A new automated upload test covers the complete transfer path.

### Installation

Install `ThorROMButler-v1.2.1.apk` over the existing installation. Settings,
custom systems, and the action log are preserved. On Android 17, allow local
network access the first time LAN receive is started.
