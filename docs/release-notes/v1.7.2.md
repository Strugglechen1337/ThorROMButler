## 🇩🇪 Deutsch

### Fehlerbehebungen (Dank an den Gerätetest!)

- **⚡ Hängender Erfolgs-Blitz behoben**: Nach dem Einsortieren blieb der goldene „einsortiert!"-Moment manchmal stehen und ließ sich nicht wegtippen — die App musste geschlossen werden. Ursache war ein Timing-Fehler, der die Weiterleitung zum Log abbrach. Der Blitz verschwindet jetzt zuverlässig von selbst und lässt sich zusätzlich durch Antippen sofort schließen.
- **💥 Absturz bei großen Ordnern behoben**: Enthielt der Ordner ein Archiv, dessen Analyse den Arbeitsspeicher sprengte (sehr große 7z/LZMA2-Kompression), beendete sich die ganze App. Solche Speicher-Fehler werden jetzt genauso abgefangen wie beim Entpacken: Das betroffene Archiv wird als fehlgeschlagen angezeigt, alle anderen laufen normal weiter.

### Installation

`ThorROMButler-v1.7.2.apk` über die bestehende Installation installieren — Einstellungen bleiben erhalten.

## 🇬🇧 English

### Bug fixes (thanks to the device test!)

- **⚡ Fixed the frozen success bolt**: after sorting, the golden "sorted in!" moment sometimes stayed on screen and couldn't be tapped away — the app had to be closed. The cause was a timing bug that cancelled the navigation to the log. The bolt now reliably dismisses itself and can also be tapped to close instantly.
- **💥 Fixed a crash on large folders**: if the folder contained an archive whose analysis exhausted memory (very large 7z/LZMA2 compression), the whole app terminated. Such memory errors are now caught the same way as during extraction: the affected archive is reported as failed and everything else continues normally.

### Install

Install `ThorROMButler-v1.7.2.apk` over the existing app — settings are kept.
